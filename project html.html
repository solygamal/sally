All modern website and web applications are built using three fundamental technology:HTML,CSSand JavaScript.These are the Languages of the web

What is HTML?

HTML stands for HyperText Markup Language.Its a markup language that web developers use to structure and describe the content of a web (not a programming language).

HTML consists of elements of that describe different types of content:paragraphs,links,video,etc,Web browsers understand HTML.andrender HTML.code as websites.

In HTML,each elements is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual elements
